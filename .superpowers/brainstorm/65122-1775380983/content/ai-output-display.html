A — Terminal log style

Output streams as monospace terminal lines with prefixes (→, ✓, $). Each line appears as the agent produces it. Feels like watching a lab instrument run. Previous runs collapsible below.

C — Collapsible result cards

Each run produces a result card (collapsed by default, expand to read). Cards show agent type, duration, status badge. Actions: Copy output, Add to Lab Notes. Run history = stack of cards.
