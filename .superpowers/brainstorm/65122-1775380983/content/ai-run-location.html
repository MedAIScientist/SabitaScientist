B — Dedicated "AI RUNS" tab in TaskDetail

TaskDetail gets two tabs: DETAILS and ⬡ AI RUNS. The AI tab shows agent picker (4 buttons), editable prompt, run history, and output — all in the same 340px drawer. Clean separation of concerns.
